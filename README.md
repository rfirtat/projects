# Projects

## 1. Data Validation Pipeline (Python + Pandas)

**Description**

This project implements a small **data validation pipeline** that reads raw transaction data from a CSV file, validates the data, removes invalid records, logs issues, and outputs a cleaned dataset.

It demonstrates how incoming data from external systems can be validated before being used downstream.

**Key Features**

* Duplicate detection and removal
* Numeric validation (quantity and price)
* Product validation against a reference dataset
* Email format validation using regex
* Date validation and parsing
* State normalization and validation
* Logging of invalid records for auditing

**Technologies Used**

* Python
* pandas
* Python logging module

**Key Skills Demonstrated**

* Data cleaning and validation
* pandas data manipulation
* Regex validation
* Data pipeline design
* Logging and traceability
